## 项目概述

这是一个数字商品自动发卡 + ChatGPT Plus 代充值平台。

- 项目路径：`/Users/siuserxiaowei/Desktop/发卡网`
- GitHub：`git@github.com:siuserxy-cmd/CardBridge.git`
- 技术栈：Node.js + Express + SQLite + 原生前端

## 当前已完成的全部功能

### 核心交易流程
- 游客下单（填邮箱，无需注册）
- 登录用户下单（JWT 认证）
- 多件购买 + 阶梯定价
- 支付宝 / 微信双支付方式（手动收款码模式）
- 支付页倒计时 + 轮询状态
- "我已支付" → 管理员确认收款 → 自动发卡
- 邮件发卡（卡密发到买家邮箱）
- 自动充值（ifaka.vip API，ChatGPT Plus CDK 充值）

### 页面
- 首页（商品列表，独立彩色卡片）
- 商品详情页（阶梯定价、购买表单、支持自动充值类商品）
- 支付页（双支付方式切换、QR 码、倒计时、轮询）
- 订单成功页 `/order/:orderNo`（直接展示卡密）
- 订单查询页 `/lookup`（邮箱 + 订单号查询）
- 管理后台 `/admin`（统计、商品、CDK 管理、订单确认/拒绝）

### 管理后台
- CDK 管理面板（批量导入、月卡/年卡分类、脱敏显示、筛选）
- 商品管理（添加/删除、交付方式设置）
- 订单管理（确认收款、拒绝、状态查看）
- 统计仪表盘（总订单、今日订单、总销售额）

## 本地启动

```bash
npm install          # 安装依赖（首次）
npm run init-db      # 初始化数据库（首次）
npm run migrate-db   # 数据库迁移（有新字段时）
npm start            # 启动服务 http://localhost:3000
```
