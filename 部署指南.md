## 本地开发

```bash
npm install          # 安装依赖
npm run init-db      # 初始化数据库（首次）
npm run migrate-db   # 迁移数据库（有新字段时）
npm start            # 启动 http://localhost:3000
npm run dev          # 开发模式（自动重启）
```

## 同步最新代码到服务器

```bash
# SSH 登录服务器
ssh root@43.224.33.197

# 进入项目目录
cd /root/cardbridge

# 拉取最新代码
git pull origin main

# 重建并启动容器
docker compose down
docker compose up -d --build

# 检查状态
server-check
```

## Docker 部署（新服务器）

1. 克隆代码
```bash
git clone git@github.com:siuserxy-cmd/CardBridge.git /root/cardbridge
cd /root/cardbridge
```

2. 配置环境变量
```bash
cp .env.example .env
nano .env  # 填写真实配置
```

3. 构建并启动
```bash
docker compose up -d --build
```

## 环境变量必填项

| 变量 | 说明 |
|------|------|
| `JWT_SECRET` | 改为随机字符串 |
| `ADMIN_EMAIL` | 管理员邮箱 |
| `ADMIN_PASSWORD` | 管理员密码（务必修改） |
| `EMAIL_USER` | QQ 邮箱地址 |
| `EMAIL_PASS` | QQ 邮箱授权码 |
| `WECHAT_PAY_QR` | 微信收款码图片路径 |
| `ALIPAY_PAY_QR` | 支付宝收款码图片路径 |

## Cloudflare Tunnel 配置

服务器上已有 `cloudflared_tunnel` 容器，Token 配置在 docker-compose 中。
域名 `api.difference-crossword.store` 已指向 Tunnel。
